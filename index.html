Fantasy Name Generator Documentation

To generate a random fantasy name:

GET to /api/generateName

Expects

Nothing

Returns

a random fantasy name in the JSON object format: { fName: "String", lName: "String" }

To generate a fantasy name based on your name:

POST to /api/generateName

Expects

a request body containing: firstName = "String", lastName = "String"

Returns

a fantasy name generated from the provided parameters in the JSON object format: { fName: "String", lName: "String" }
